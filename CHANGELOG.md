### Fixed
- Minor bug fixes and improvements

## [1.2.0] - Previous Release

### Added
- Initial release of http-client-with-recaptcha
- Angular service for HTTP client with reCAPTCHA v3 integration
- Support for customizable header names
- Seamless integration with ng-recaptcha

### Features
- Easy integration with Angular applications
- Configurable reCAPTCHA token header
- Support for all HTTP methods
- Automatic token management
